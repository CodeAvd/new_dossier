Mono pill chip — tech-stack labels, evidence metrics, categories, statuses. Lowercase, terse.

```jsx
<Tag tone="accent">python</Tag>
<Tag tone="sage" dot>shipped</Tag>
<Tag tone="honey">+1.8 sharpe</Tag>
<Tag tone="outline">2024</Tag>
```

Tones: `accent` (default), `neutral`, `outline`, `spark` (vivid cyan — the cool pop), the pastels `blush` / `honey` / `sage` / `sky` / `mauve`, and semantic `success` / `warning` / `danger` / `info`. Set `dot` for a leading status dot.
