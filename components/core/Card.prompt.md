Soft warm surface container — the base for project cards, panels, and wells.

```jsx
<Card padding="lg" interactive>
  <h3>agent that cites its evidence</h3>
  <p>file-first retrieval, every claim linked to a source span.</p>
</Card>

<Card elevation="sunken" padding="md">code block / quiet well</Card>
```

`elevation`: `flat` / `default` / `raised` / `sunken` / `dashed` (playful paper outline, david-hckh.com motif) / `holo` (deep holographic-blue panel for quant/data). `padding`: `none` / `sm` / `md` / `lg`. Set `interactive` for hover lift.
